## 📖 使用说明

### 基本使用
1. **输入演讲主题**：在左侧输入框中填写主题
2. **设置参数**：选择字数、风格、场合等
3. **选择AI模型**：系统会智能选择最优模型
4. **生成演讲稿**：点击"🚀 生成演讲稿"按钮
5. **精细优化**：使用二次优化面板进行专业优化

### 🤖 AI模型说明
| 模型 | 特点 | 推荐用途 |
|------|------|----------|
| **GPT-4** | 逻辑严密，质量最高 | 重要演讲 |
| **Claude Opus** | 深度思考，安全可靠 | 专业分析 |
| **DeepSeek** | 创新表达，成本优化 | 日常使用 |
| **通义千问** | 中文优化，本土化强 | 中文内容 |

### 🎨 优化系统
二次优化面板提供8种专业优化维度：

#### 📝 内容优化
- 🧠 **逻辑结构优化**：使论证更加严密，层次更加清晰
- 📚 **内容深度增强**：补充专业见解和深层分析

#### 🎭 表达优化
- ❤️ **情感感染力**：提升感染力和共鸣度
- 🎪 **修辞技巧润色**：运用排比、递进等手法

#### 📊 专业优化
- 📈 **数据案例补强**：增加具体数据和成功案例
- 🔬 **专业术语精准**：确保术语使用的准确性

#### 🎯 风格优化
- 👨‍💼 **杨晖风格强化**：突出技术专家身份特色
- 🏆 **权威性提升**：体现行业地位和专业影响力
