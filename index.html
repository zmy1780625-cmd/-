一、云计算与AI基础设施
01
FlexKV 分布式KV缓存架构合入多个开源推理框架

腾讯云主导的 FlexKV 开源项目近期合入 vLLM 等主流推理框架。该技术采用多级缓存方案，旨在扩展推理场景下的可用 KV Cache 容量，降低高并发推理时的显存压力与单位 Token 成本。开发者可在支持框架中配置启用。

🔥 开源里程碑推理优化KV Cache
02
腾讯混元联合Robotics X发布具身智能模型系列

混元团队联合 Robotics X 发布面向具身智能场景的模型系列，提供 2B 端侧轻量版和 32B 旗舰版两个规格。模型采用混合 Transformer 架构与自研视觉编码器，在多项评测中表现较好。目前已在机器人实机控制场景中进行了初步验证。

🔥 开源里程碑具身智能混元模型
03
CLS 发布 Cloud Mate 智能运维 Agent

CLS 日志服务推出 Cloud Mate 智能运维 Agent，可对业务指标、日志、链路追踪及变更记录进行关联分析，在告警触发时辅助运维人员进行根因定位。支持 CLS、Prometheus 等监控平台接入。

智能运维云原生可观测性
04
ClawPro 联合 Agent Storage 推出智能体数据空间功能

ClawPro 联合腾讯云存储 Agent Storage 推出数据空间功能，为 AI 智能体提供独立的文件存储与管理能力。支持数据隔离、跨端访问、文件上传备份及下载链接生成。

ClawPro云存储数据空间

腾讯云推出 AI 助手 LightClaw ACE，面向云上场景设计。具备三层记忆管理、上下文压缩等能力，内置安全机制与云原生技能，可通过轻量应用服务器快速部署，支持接入 DeepSeek、OpenAI 等大模型。

LightClawAI助手云原生

浏览器AI语音交互自动化

Buddy 系列产品模型升级

WorkBuddy、CodeBuddy IDE、插件及 CLI 等 Buddy 系列产品的内置模型已升级至新版本。新版模型在代码生成、任务规划等基准测试中表现提升，支持较长任务的自主执行。

三、研发效能与项目管理
10
TAPD 推出内嵌式 AI 项目管理助手

TAPD 推出内嵌式 AI 项目管理助手 NPC，支持异步后台执行。功能包括需求拆分、依赖冲突检测、验收标准生成、进度审计、工时分析、自动化周报月报生成等。据官方数据，部分场景效率有较明显提升。

🔥 效率革命TAPD项目管理
11
CodeBuddy 集成 CloudQ 技能实现自动化巡检

CodeBuddy集成CloudQ技能实现自动化云上巡检治理。用户通过自然语言设定定时任务（如"每周一早9点发巡检报告"），系统自动拉取监控数据、生成可视化报告并推送群聊。支持STS或AK/SK授权，AI可精准捕捉接口响应时间增长15%等潜在风险，传统20分钟人工巡检流程实现全自动化。

CodeBuddy智能巡检自动化运维
12
腾讯会议 AI 能力赋能同桌教育课程质检

在线教育机构同桌教育借助腾讯会议AI能力实现100%课程质检覆盖。核心技术包括实时语音转文字集成教师端、课后AI自动质检多维度异常扫描、AI小助手Pro支持自然语言分析课堂质量，以及智能章节定位与会议数据API可视化预警。2名质检人员即可完成月均2万节课的精查工作，外教从"打字速记"解放为互动"教练"。

腾讯会议AI质检教育场景

腾讯问卷联合电子签推出签署功能

腾讯问卷联合腾讯电子签推出"签署"功能，将信息采集与合同签署合二为一。底层打通问卷数据与电子合同数据映射，采用实名认证（人脸识别）确保签署主体真实，系统自动生成具备法律效力的签署凭证PDF。传统5步流程压缩为2步，覆盖活动报名、业主投票、满意度评价、入职登记等4大典型场景。

本周总结与分析

本周技术动态围绕三条主线展开：一是 Agent 能力持续拓展。QClaw 多 Agent 协同、TAPD NPC 项目管理助手、LightClaw ACE 云原生助手等产品的更新，表明腾讯正在构建覆盖办公、开发、运维、浏览器等场景的 Agent 产品矩阵。二是模型与推理基础设施稳步推进。FlexKV 合入多个开源推理框架，降低推理成本；新版模型在 Buddy 全系及 ima 中快速落地；混元具身智能模型面向机器人场景展开验证。三是 AI 与行业场景进一步结合。腾讯会议 AI 辅助教育质检、腾讯问卷签署功能整合信息采集与合同流程、乐享自定义 Skills 扩展知识库 Agent 能力，反映出 AI 正从技术工具向业务生产力转化。
